
# Firebase Studio - Bangla Naturals E-commerce Platform

এটি একটি আধুনিক মাল্টি-টিন্যান্ট ই-কমার্স প্ল্যাটফর্ম।

## সিকিউরিটি এবং ডাটাবেস সেটআপ (Security Fixes)

সুপাবেস (Supabase) ড্যাশবোর্ডে RLS, Function Search Path বা ইনসিকিউর পলিসি নিয়ে কোনো ওয়ার্নিং দেখলে নিচের ধাপগুলো অনুসরণ করুন:

১. আপনার **Supabase Dashboard**-এ যান।
২. বাম পাশের মেনু থেকে **SQL Editor** সেকশনে যান।
৩. এই প্রকল্পের রুট ডিরেক্টরিতে থাকা `supabase_security_fixes.sql` ফাইলটি ওপেন করুন।
৪. ভেতরের কোডটুকু কপি করে সুপাবেস এডিটরে পেস্ট করুন এবং **Run** বাটনে ক্লিক করুন।

### এই ফিক্সগুলো যা নিশ্চিত করে:
- **Function Security**: `update_updated_at_column`, `trigger_set_timestamp` সহ সকল ফাংশন এখন "Search Path Hijacking" আক্রমণ থেকে সুরক্ষিত।
- **Secure `uncompleted_orders`**: অনিরাপদ আপডেট এবং ডিলিট পলিসি সরিয়ে ফেলা হয়েছে। এখন এটি সম্পূর্ণ সুরক্ষিত।
- **Secure `orders`**: সরাসরি পাবলিক ইনসার্ট বন্ধ করা হয়েছে। এখন শুধুমাত্র অনুমোদিত এপিআই-এর মাধ্যমে অর্ডার গ্রহণ করা হবে।
- **Secure `customer_addresses`**: গ্রাহকদের ব্যক্তিগত ঠিকানা এখন সম্পূর্ণ সুরক্ষিত এবং প্রাইভেট।
- **Secure `live_chat_messages`**: চ্যাট মেসেজ ইনসার্ট এবং অ্যাডমিন এক্সেস এখন সঠিক মালিকানা (site_id) অনুযায়ী নিয়ন্ত্রিত।
- **Optimized `carousel_slides`**: ডুপ্লিকেট পলিসি মুছে ফেলে পারফরম্যান্স বাড়ানো হয়েছে।

## শুরু করার নিয়ম

অ্যাপটি ডেভেলপমেন্ট মোডে রান করতে `npm run dev` কমান্ডটি ব্যবহার করুন। মূল কোড দেখতে `src/app/page.tsx` ফাইলটি দেখুন।
