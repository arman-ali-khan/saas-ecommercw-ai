
# Firebase Studio - Bangla Naturals E-commerce Platform

এটি একটি আধুনিক মাল্টি-টিন্যান্ট ই-কমার্স প্ল্যাটফর্ম।

## সিকিউরিটি এবং ডাটাবেস সেটআপ (Security Fixes)

সুপাবেস (Supabase) ড্যাশবোর্ডে RLS বা Function Search Path নিয়ে কোনো ওয়ার্নিং বা এরর দেখলে নিচের ধাপগুলো অনুসরণ করুন:

১. আপনার **Supabase Dashboard**-এ যান।
২. বাম পাশের মেনু থেকে **SQL Editor** সেকশনে যান।
৩. এই প্রকল্পের রুট ডিরেক্টরিতে থাকা `supabase_security_fixes.sql` ফাইলটি ওপেন করুন।
৪. ভেতরের কোডটুকু কপি করে সুপাবেস এডিটরে পেস্ট করুন এবং **Run** বাটনে ক্লিক করুন।

### এই ফিক্সগুলো যা নিশ্চিত করে:
- **Optimized `carousel_slides`**: ডুপ্লিকেট পলিসি মুছে ফেলে পারফরম্যান্স উন্নত করা হয়েছে।
- **Function Security**: `handle_updated_at` এবং `handle_page_updated_at` ফাংশনগুলোকে সুরক্ষিত করা হয়েছে।
- **Secure `customer_addresses`**: গ্রাহকদের ব্যক্তিগত ঠিকানা এখন সম্পূর্ণ সুরক্ষিত।
- **Secure `live_chat_messages`**: চ্যাট মেসেজ পাঠানোর সময় ডেটা ভ্যালিডেশন নিশ্চিত করা হয়েছে।
- **Secure `saas_reviews`**: ভুয়া রিভিউ সরাসরি পাবলিশ হওয়া বন্ধ করা হয়েছে।
- **Secure `orders`**: অর্ডারের তথ্য এখন শুধুমাত্র এপিআই-এর মাধ্যমে নিয়ন্ত্রিত।

## শুরু করার নিয়ম

অ্যাপটি ডেভেলপমেন্ট মোডে রান করতে `npm run dev` কমান্ডটি ব্যবহার করুন। মূল কোড দেখতে `src/app/page.tsx` ফাইলটি দেখুন।
